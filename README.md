# gloomy
Some kind of safer OpenGL wrappers and convenience namespace.

### Build
```sh
> mkdir build
> cd build
> cmake ..
> make # or msbuild yadda yadda if you're using Visual Studio.
```

### Examples
Check `learnopengl` directory.
